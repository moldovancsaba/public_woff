# AS Roma Font Host

Next.js site for hosting WOFF fonts publicly.

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Import to Vercel (Framework: Next.js)
3. Deploy

## Add Fonts

Place your `.woff` files in `public/fonts/` and they'll be accessible at `/fonts/your-font.woff`
